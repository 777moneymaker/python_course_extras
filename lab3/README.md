# G/C Percentage
Simple scripts written as an extra tasks.
Written for AMU's Python course.
Simple script to open 10 files from data folder and count G/C bases percentage for each file.

Usage
-----
Standalone
```
python3 script.py
# or
chmod +x script.py
# then
./script.py
```
